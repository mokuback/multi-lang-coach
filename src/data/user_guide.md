# 歡迎使用 Multi-Lang Coach！🚀

這是一個專為 IT 專業人士與商務人士設計的「大數據驅動語音對話教練」。
在這裡，您可以根據您的職務與每日任務，練習**英文**或**日文**的職場對話，並有效累積您的專屬生詞與實用句型！

---

## 🔑 第一步：設定與 API 啟用 (非常重要)

預設情況下，系統使用「模擬對話體驗模式」來展示基本流程。若要解鎖強大的 AI 功能，請依照以下步驟設定：

1. 點擊左側選單最下方的 **[設定與 API]**。
2. 在「Gemini API 金鑰」欄位中填入您的專屬 Key。
   - *(若沒有金鑰，請前往 [Google AI Studio](https://aistudio.google.com/) 免費申請)*
3. 在同一頁面中，您也可以設定您的**主類別**、**職務**、**語言程度**、以及您想要學習的**目標語言**。
4. 設定完成後點擊「儲存設定並返回」。

---

## 🗺️ 第二步：開始您的學習旅程

本系統的設計圍繞著一個高效的學習循環：**「任務選取 ➔ 實戰對話 ➔ 知識解析 ➔ 複習與匯出」**。

### 1. 每日任務 (Dashboard)
- 進入 **[每日任務]**，系統會根據您的職務，列出 6 個不同難度的情境任務（從簡單回報到困難的技術談判）。
- 點擊「開始對話練習」進入實戰。

### 2. 對話練習模式 (Chat Practice)
- **口說輸入**：點擊「麥克風🎙️」按鈕，直接用語音說出您的回覆。
- **自動朗讀**：AI 回覆後，系統會自動朗讀（可在設定中關閉）。若想再次聆聽，可點擊每則訊息下方的「朗讀」按鈕。
- **💡 本情境推薦句型**：在聊天視窗上方，您可以展開面板，查看適合該情境的母語人士常用句型，試著把它們用在您的回覆中！
- **✨ AI 潤飾建議**：輸入草稿（甚至是中文），點擊「魔法棒」，AI 會根據對話脈絡幫您翻譯或修飾成最道地的說法。
- **🔍 當句解析**：遇到看不懂的 AI 回覆？點擊「當句解析」，AI 會詳細拆解文法，並自動為您萃取單字與句型！

### 3. 生詞與句型筆記本 (Notebook)
- 在對話中被 AI 糾正的單字，或是透過「當句解析」萃取出的精華，都會自動收錄於左側的 **[生詞筆記本]** 與 **[句型筆記本]** 中。
- 您可以隨時點擊發音按鈕複習，或是點擊「匯出」將它們打包成 PDF 檔列印下來。

### 4. 常用句型總覽 (Patterns)
- 進入 **[常用句型]** 頁面，系統會為您彙整當前情境的所有精選句型。
- **版本切換**：您可以在上方自由切換「基礎教科書版 (01)」與「進階商務版 (02)」，體驗不同層次的表達方式。
- **代換練習**：點擊任何一個句型卡片上的「進入代換練習」，系統會啟動教練模式，給予您中文情境，訓練您將單字套入句型中靈活運用！

---

## 💡 進階技巧
- **選取文字解析**：在對話過程中，您可以直接「反白選取」AI 訊息中的某個特定單字或片語，系統會跳出「標記解析」按鈕，讓您針對該字詞進行深入學習。
- **文法糾錯嚴格度**：若您只是想練膽量，可以在設定中選擇「溝通為主」；若您即將面對重要面試或簡報，可切換至「超級嚴格」模式，AI 將會抓出您所有細微的不自然表達。

> 現在，就去 **[設定與 API]** 輸入您的金鑰，前往 **[每日任務]** 挑選一個情境，開始您的第一場對話練習吧！
