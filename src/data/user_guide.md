# 歡迎使用 Multi-Lang Coach！🚀

這是一個專為 IT 專業人士與商務人士設計的「大數據驅動語音對話教練」。
在這裡，您可以根據您的職務與每日任務，練習**英文**或**日文**的職場對話，並有效累積您的專屬生詞與實用句型！

---

## 🔑 第一步：設定與 API 啟用 (非常重要)

預設情況下，系統使用「模擬對話體驗模式」來展示基本流程。若要解鎖強大的 AI 功能，請依照以下步驟設定：

1. 點擊左側選單最下方的 **[設定與 API]**。
2. 在「Gemini API 金鑰」欄位中填入您的專屬 Key。
   - *(若沒有金鑰，請前往 [Google AI Studio](https://aistudio.google.com/) 免費申請)*
3. 在同一頁面中，您也可以設定您的**主類別**、**職務**、**語言程度**、以及您想要學習的**目標語言**。
4. 設定完成後點擊「儲存設定並返回」。

---

## 🗺️ 第二步：開始您的學習旅程

本系統的設計圍繞著一個高效的學習循環：**「任務選取 ➔ 實戰對話 ➔ 知識解析 ➔ 複習與匯出」**。

### 1. 每日任務 (Dashboard)
- 進入 **[每日任務]**，系統會根據您的職務，列出 6 個不同難度的「情境任務」（從簡單的工作回報到困難的技術談判）。
- **開始練習**：點選任一卡片上的「開始練習」，即可進入專屬該情境的對話。
- **✨ 自由對話模式 (Free Mode)**：不想受到主題限制？點擊任務列表最上方的 **「✨ 自由對話模式」**，即可隨心所欲地跟 AI 閒聊近況或探討最新趨勢，訓練最真實的臨場反應！

### 2. 對話練習模式 (Chat Practice)
- **口說輸入**：點擊「麥克風🎙️」按鈕，直接用語音說出您的回覆。
- **自動朗讀**：AI 回覆後，系統會自動朗讀（可在設定中關閉）。若想再次聆聽，可點擊每則訊息下方的「朗讀」按鈕。
- **💡 本情境推薦句型**：在聊天視窗上方，您可以展開面板，查看適合該情境的母語人士常用句型，試著把它們用在您的回覆中！（*註：自由對話模式下將自動隱藏此面板*）
- **✨ AI 潤飾建議**：即使是用中文構思，或是輸入不完整的單字草稿，只要點擊對話框旁的「魔法棒🪄」，AI 就會根據上下文，幫您翻譯或修飾成最道地的母語說法。
- **🔍 當句解析與單字萃取**：遇到看不懂的 AI 回覆？點擊訊息下方的「當句解析」，AI 會詳細拆解句型文法，並自動為您萃取關鍵單字，收錄至筆記本！

### 3. 生詞與句型筆記本 (Notebook)
- 在對話中被 AI 糾正的單字，或是透過「當句解析」萃取出的精華，都會自動收錄於左側的 **[生詞筆記本]** 與 **[句型筆記本]** 中。
- 您可以隨時點擊發音按鈕複習，或是點擊「匯出」將它們打包成 PDF 檔列印下來。

> [!WARNING]
> **⚠️ 資料儲存注意事項**：您的筆記資料是儲存在您的「瀏覽器本地端 (Local Storage)」。如果清除了瀏覽器的 Cookie 或快取資料，筆記將會遺失！建議您**定期使用「匯出」功能**，將辛苦收集的生詞與句型備份為 PDF 檔案。

### 4. 常用句型總覽 (Patterns)
- 進入 **[常用句型]** 頁面，系統會為您彙整當前情境的所有精選句型。
- **版本切換**：您可以在上方自由切換「基礎教科書版 (01)」與「進階商務版 (02)」，體驗不同層次的表達方式。
- **代換練習**：點擊任何一個句型卡片上的「進入代換練習」，系統會啟動教練模式，給予您中文情境，訓練您將單字套入句型中靈活運用！

---

## 💡 進階技巧
- **選取文字解析**：在對話過程中，您可以直接「反白選取」AI 訊息中的某個特定單字或片語，系統會跳出「標記解析」按鈕，讓您針對該字詞進行深入學習。
- **文法糾錯嚴格度**：若您只是想練膽量，可以在設定中選擇「溝通為主」；若您即將面對重要面試或簡報，可切換至「超級嚴格」模式，AI 將會抓出您所有細微的不自然表達。

## 📱 將網站安裝為手機 App (免下載)

本系統支援響應式設計，為了獲得最佳的操作體驗（全螢幕、無瀏覽器網址列干擾），建議您將本網站加入手機的主畫面：

### 🍎 iOS (iPhone / iPad) 使用者：
1. 使用 **Safari** 瀏覽器開啟本網站。
2. 點擊畫面底部的 **「分享 ⬆️」** 圖示。
3. 往下滑動，選擇 **「加入主畫面 (Add to Home Screen)」**。
4. 點擊右上角的「新增」，桌面上就會出現專屬的 App 圖示了！

### 🤖 Android 使用者：
1. 使用 **Chrome** 瀏覽器開啟本網站。
2. 點擊畫面右上角的 **「設定 (三個點 ⋮)」** 圖示。
3. 選擇 **「加入主畫面」** 或 **「安裝應用程式」**。
4. 確認新增後，即可在手機桌面上像一般 App 一樣直接開啟！
