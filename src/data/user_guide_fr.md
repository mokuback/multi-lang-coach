# Bienvenue sur Multi-Lang Coach ! 🚀

Il s'agit d'un "Entraîneur de Conversation Vocale Piloté par le Big Data" conçu spécifiquement pour les professionnels de l'informatique et les gens d'affaires.
Ici, vous pouvez pratiquer des conversations sur le lieu de travail en **anglais** ou en **japonais** en fonction de votre poste et de vos tâches quotidiennes, en accumulant efficacement votre vocabulaire personnel et vos modèles pratiques !

---

## 🔑 Étape 1 : Paramètres et Activation de l'API (Très Important)

Par défaut, le système utilise le "Mode d'Expérience de Dialogue Simulé" pour démontrer le flux de base. Pour débloquer de puissantes fonctionnalités d'IA, veuillez suivre ces étapes pour configurer :

1. Cliquez sur **[Paramètres et API]** en bas du menu de gauche.
2. Sélectionnez le **Fournisseur d'IA** que vous souhaitez utiliser et entrez la clé API et le Nom du Modèle (ID du Modèle) correspondants :

> **💎 Option A : Google Gemini (Recommandation par Défaut)**
> 1. Allez sur [Google AI Studio](https://aistudio.google.com/app/apikey) pour vous connecter.
> 2. Cliquez sur **"Create API key"** > **"Create API key in new project"**.
> 3. Copiez la chaîne commençant par **AIza** et collez-la dans ce système.
> *(Remarque : l'API Gemini offre actuellement un quota gratuit généreux pour les utilisateurs généraux, aucune carte de crédit requise !)*

> **⚡ Option B : Groq (Expérience de Réponse Ultra-rapide)**
> 1. Allez sur [GroqConsole](https://console.groq.com/keys) pour vous connecter.
> 2. Cliquez sur **"Create API Key"**.
> 3. Copiez la clé générée `gsk_...` et collez-la dans le système.
> 4. **Nom du Modèle (ID du Modèle)** : assurez-vous de remplir le bon nom de modèle. La recommandation par défaut est `llama-3.1-8b-instant` ou `llama-3.3-70b-versatile`, ce qui apporte une expérience extrêmement rapide !

3. Sur la même page, vous pouvez également définir votre **Catégorie Principale**, **Sujets et Métiers**, **Niveau de Langue** et votre **Langue Cible**.
4. Après la configuration, cliquez sur "Enregistrer et Retourner".

---

## 🗺️ Étape 2 : Commencez votre Voyage d'Apprentissage

La conception de ce système s'articule autour d'un cycle d'apprentissage efficace : **"Sélection des Tâches ➔ Pratique de Conversation ➔ Analyse des Connaissances ➔ Révision et Exportation"**.

### 1. Tâches Quotidiennes (Dashboard)
- Entrez dans **[Tâches Quotidiennes]**, et le système listera 6 "Tâches de Scénario" de difficultés variables en fonction de votre rôle (des simples rapports de travail aux négociations techniques difficiles).
- **Commencer la Pratique** : Cliquez sur "Commencer la Pratique" sur n'importe quelle carte pour entrer dans la conversation dédiée à ce scénario.
- **✨ Mode de Conversation Libre** : Vous ne voulez pas être limité par des sujets ? Cliquez sur **"✨ Mode de Conversation Libre"** en haut de la liste des tâches pour discuter librement avec l'IA d'événements récents ou discuter des dernières tendances, en entraînant vos réactions sur le moment les plus authentiques !

### 2. Mode de Pratique de Conversation
- **Entrée Vocale** : Cliquez sur le bouton "Microphone 🎙️" pour parler votre réponse directement avec votre voix.
- **Lecture Automatique** : Après la réponse de l'IA, le système la lira à voix haute automatiquement (peut être désactivé dans les Paramètres). Si vous voulez réécouter, cliquez sur le bouton de lecture sous chaque message.
- **💡 Modèles Recommandés** : Au-dessus de la fenêtre de discussion, vous pouvez développer un panneau pour afficher les modèles courants utilisés par les locuteurs natifs adaptés au scénario. Essayez de les utiliser dans vos réponses ! (*Remarque : ce panneau est automatiquement masqué dans le Mode de Conversation Libre*)
- **✨ Suggestions de l'IA** : Même si vous réfléchissez dans votre langue maternelle ou si vous entrez des brouillons de mots incomplets, cliquez simplement sur la "Baguette Magique 🪄" à côté de la zone de discussion, et l'IA traduira ou peaufinera l'expression la plus native en fonction du contexte.
- **🔍 Analyse de Phrase et Extraction de Vocabulaire** : Vous rencontrez une réponse de l'IA que vous ne comprenez pas ? Cliquez sur "Analyse de Phrase" sous le message, et l'IA décomposera la grammaire en détail et extraira automatiquement le vocabulaire clé pour vous, en l'enregistrant dans votre carnet !
- **📥 Exporter et Évaluation de l'IA** : Après avoir eu une conversation avec l'IA, cliquez sur le bouton "Exporter" en haut de l'écran. Vous pouvez choisir d'exporter l'historique de discussion directement au format PDF, ou choisir "Exporter et Demander l'Analyse IA" pour demander à l'IA d'évaluer vos performances globales à l'oral, votre grammaire et votre ton, et de vous fournir des conseils récapitulatifs !

### 3. Carnet de Vocabulaire et de Modèles
- Le vocabulaire corrigé par l'IA pendant les conversations, ou l'essence extraite via "Analyse de Phrase", sera automatiquement enregistré dans votre **[Carnet de Vocabulaire]** et votre **[Carnet de Modèles]** à gauche.
- Vous pouvez cliquer sur le bouton de prononciation pour réviser à tout moment, ou cliquer sur "Exporter" pour les regrouper dans un fichier PDF et les imprimer.

> [!WARNING]
> **⚠️ Avis de Stockage des Données** : Les données de votre carnet sont stockées dans le "stockage local de votre navigateur". Si vous effacez les cookies ou les données de cache du navigateur, vos notes seront perdues ! Il est recommandé d'**utiliser régulièrement la fonction "Exporter"** pour sauvegarder le vocabulaire et les modèles que vous avez minutieusement collectés sous forme de fichiers PDF.

### 4. Modèles Communs
- Entrez dans la page **[Modèles Communs]**, et le système compilera pour vous tous les modèles sélectionnés pour le scénario actuel.
- **Changement de Version** : Vous pouvez basculer librement entre la "Version Manuels de Base" et la "Version Entreprise Avancée" en haut pour découvrir différents niveaux d'expression.
- **Pratique de Substitution** : Cliquez sur "Pratique de Substitution" sur n'importe quelle carte de modèle, et le système activera le mode coach, vous donnera un scénario natif et vous entraînera à appliquer des mots de manière flexible dans le modèle !

## 💡 Conseils Avancés
- **Sélectionner le Texte à Analyser** : Pendant la conversation, vous pouvez directement "surligner et sélectionner" un mot ou une phrase spécifique dans un message de l'IA. Le système affichera un bouton "Analyse Marquée", vous permettant d'étudier ce terme spécifique en profondeur.
- **Rigueur Grammaticale** : Si vous voulez juste pratiquer le courage de parler, vous pouvez choisir "Communication Prioritaire (Corriger seulement les erreurs graves)" dans les paramètres ; si vous êtes sur le point de faire face à un entretien ou à une présentation importante, passez en mode "Super Strict (Détecter toutes les imperfections)", et l'IA détectera toutes vos expressions subtiles non naturelles.

## 📱 Installer le Site Web comme Application Mobile (Aucun Téléchargement Requis)

Ce système prend en charge la conception réactive. Pour obtenir la meilleure expérience utilisateur (plein écran, aucune interférence de la barre d'adresse du navigateur), nous vous recommandons d'ajouter ce site Web à l'écran d'accueil de votre téléphone :

### 🍎 Utilisateurs iOS (iPhone / iPad) :
1. Ouvrez ce site Web à l'aide du navigateur **Safari**.
2. Cliquez sur l'icône **"Partager ⬆️"** en bas de l'écran.
3. Faites défiler vers le bas et sélectionnez **"Sur l'écran d'accueil"**.
4. Cliquez sur "Ajouter" dans le coin supérieur droit, et une icône d'application dédiée apparaîtra sur votre écran d'accueil !

### 🤖 Utilisateurs Android :
1. Ouvrez ce site Web à l'aide du navigateur **Chrome**.
2. Cliquez sur l'icône **"Paramètres (Trois points ⋮)"** dans le coin supérieur droit de l'écran.
3. Sélectionnez **"Ajouter à l'écran d'accueil"** ou **"Installer l'application"**.
4. Après avoir confirmé l'ajout, vous pouvez l'ouvrir directement depuis l'écran d'accueil de votre téléphone comme une application normale !
