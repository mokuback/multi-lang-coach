# Multi-Lang Coach へようこそ！🚀

これは、IT プロフェッショナルやビジネスパーソンのために特別に設計された「ビッグデータ主導の音声会話コーチ」です。
ここでは、あなたの職務や日替わりタスクに基づいて、**英語** または **日本語** の職場での会話を練習し、専用の単語や実用的な文法を効果的に蓄積することができます！

---

## 🔑 ステップ 1: 設定 & API の有効化 (非常に重要)

デフォルトでは、基本的な流れを示すために「モック会話体験モード」が使用されています。強力な AI 機能を解放するには、以下の手順に従って設定を行ってください：

1. 左側のメニューの一番下にある **[設定 & API]** をクリックします。
2. 使用したい **AI プロバイダー** を選択し、対応する API キーとモデル名を入力します：

> **💎 オプション A: Google Gemini (デフォルト推奨)**
> 1. [Google AI Studio](https://aistudio.google.com/app/apikey) にログインします。
> 2. **「Create API key」** > **「Create API key in new project」** をクリックします。
> 3. **AIza** で始まる文字列をコピーし、このシステムに貼り付けます。
> *(注: Gemini API は現在、一般ユーザーに寛大な無料枠を提供しており、クレジットカードの登録は不要です！)*

> **⚡ オプション B: Groq (超高速レスポンス体験)**
> 1. [GroqConsole](https://console.groq.com/keys) にログインします。
> 2. **「Create API Key」** をクリックします。
> 3. 生成されたキー `gsk_...` をコピーしてシステムに貼り付けます。
> 4. **モデル名 (Model ID)**: 正しいモデル名が入力されていることを確認してください。デフォルトの推奨は `llama-3.1-8b-instant` または `llama-3.3-70b-versatile` で、これにより超高速な体験が得られます！

3. 同じページで、**メインカテゴリ**、**トピックと役割**、**言語レベル**、および学習したい **目標言語** を設定することもできます。
4. 設定が完了したら、「保存して戻る」をクリックします。

## 🗺️ ステップ 2: 学習の旅を始めましょう

このシステムの設計は、「タスク選択 ➔ 会話練習 ➔ 知識分析 ➔ 復習とエクスポート」という効率的な学習サイクルを中心に展開されています。

### 1. 日替わりタスク (Dashboard)
- **[日替わりタスク]** に入ると、システムはあなたの役割に基づいて、難易度の異なる 6 つの「シナリオタスク」をリストアップします（簡単な業務報告から困難な技術交渉まで）。
- **練習開始**: 任意のカードの「練習開始」をクリックすると、そのシナリオ専用の会話に入ります。
- **✨ フリートークモード**: トピックに制限されたくないですか？ タスクリストの一番上にある **「✨ フリートークモード」** をクリックすると、近況についてAIと自由にチャットしたり、最新のトレンドについて話し合ったりして、最もリアルな臨場感あふれる反応を訓練できます！

### 2. 会話練習モード
- **音声入力**: 「マイク 🎙️」ボタンをクリックして、音声で直接返答を話します。
- **自動読み上げ**: AI が返答した後、システムは自動的に読み上げます（設定でオフにできます）。もう一度聞きたい場合は、各メッセージの下にある「読み上げ」ボタンをクリックします。
- **💡 推奨文法**: チャットウィンドウの上部でパネルを展開し、そのシナリオに適したネイティブスピーカーがよく使う文法を確認できます。ぜひ返答で使ってみてください！（*注：フリートークモードでは、このパネルは自動的に非表示になります*）
- **✨ AI 改善提案**: 母国語で考えたり、不完全な単語のドラフトを入力したりした場合でも、チャットボックスの横にある「魔法の杖 🪄」をクリックするだけで、AI が文脈に基づいて最もネイティブな表現に翻訳または改善してくれます。
- **🔍 文を分析 & 単語抽出**: 理解できない AI の返答に遭遇しましたか？ メッセージの下にある「文を分析」をクリックすると、AI が文法を詳細に分解し、重要な単語を自動的に抽出してノートブックに保存します！
- **📥 エクスポート & AI 評価**: AI との会話を終えた後、画面上部の「エクスポート」ボタンをクリックします。会話の履歴をそのまま PDF としてエクスポートするか、「エクスポート & 分析」を選択して、AI にスピーキングのパフォーマンス全体、文法、トーンを評価させ、まとめのアドバイスをもらうことができます！

### 3. 単語と文法ノートブック
- 会話中に AI によって修正された単語、または「文を分析」を通じて抽出されたエッセンスは、左側の **[単語ノート]** および **[文法ノート]** に自動的に保存されます。
- 発音ボタンをクリックしていつでも復習したり、「エクスポート」をクリックして PDF ファイルにまとめて印刷したりできます。

> [!WARNING]
> **⚠️ データ保存に関する注意**: ノートブックのデータは「ブラウザのローカルストレージ」に保存されます。ブラウザの Cookie やキャッシュデータをクリアすると、ノートが失われます！ 一生懸命集めた単語や文法を PDF ファイルとしてバックアップするために、**定期的に「エクスポート」機能を使用する**ことをお勧めします。

### 4. よく使う文法
- **[よく使う文法]** ページに入ると、システムは現在のシナリオに選択されたすべての文法をまとめてくれます。
- **バージョン切り替え**: 上部で「基礎教科書版 (初級者向け)」と「高度なビジネス版 (ネイティブの思考)」を自由に切り替えて、異なるレベルの表現を体験できます。
- **置き換え練習**: 任意の文法カードの「置き換え練習」をクリックすると、システムはコーチモードを起動し、ネイティブなシナリオを提示して、単語を文法に当てはめて柔軟に活用できるように訓練します！

## 💡 高度なヒント
- **選択したテキストを分析**: 会話中、AI のメッセージ内の特定の単語やフレーズを直接「ハイライトして選択」することができます。システムは「分析済み」ボタンをポップアップ表示し、その特定の用語を深く学習できるようにします。
- **文法修正の厳密さ**: スピーキングの度胸を鍛えたいだけなら、設定で「コミュニケーション重視」を選択できます。重要な面接やプレゼンテーションを控えている場合は、「超厳密（すべてのエラーを修正）」モードに切り替えると、AI がすべてのわずかに不自然な表現をキャッチします。

## 📱 ウェブサイトをモバイルアプリとしてインストール (ダウンロード不要)

このシステムはレスポンシブデザインをサポートしています。最高のエクスペリエンス（フルスクリーン、ブラウザのアドレスバーの干渉なし）を得るために、このウェブサイトをスマートフォンのホーム画面に追加することをお勧めします：

### 🍎 iOS (iPhone / iPad) ユーザー：
1. **Safari** ブラウザを使用してこのウェブサイトを開きます。
2. 画面下部の **「共有 ⬆️」** アイコンをクリックします。
3. 下にスクロールして、**「ホーム画面に追加」** を選択します。
4. 右上の「追加」をクリックすると、専用のアプリケーショアイコンがホーム画面に表示されます！

### 🤖 Android ユーザー：
1. **Chrome** ブラウザを使用してこのウェブサイトを開きます。
2. 画面右上の **「設定 (3つの点 ⋮)」** アイコンをクリックします。
3. **「ホーム画面に追加」** または **「アプリをインストール」** を選択します。
4. 追加を確認すると、通常のアプリと同じようにスマートフォンのホーム画面から直接開くことができます！

> **🎉 始める準備はできましたか？** 
> 
> まずは **[設定 & API]** に移動してキーを入力し、役割を設定してください。次に **[日替わりタスク]** に移動してシナリオを選択するか、**フリートークモード** を開始して、あなた専用の外国語コーチをお楽しみください！
