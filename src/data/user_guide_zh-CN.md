# 欢迎使用 Multi-Lang Coach！🚀

这是一个专为 IT 专业人士与商务人士设计的"大数据驱动语音对话教练"。
在这里，您可以根据您的职务与每日任务，练习**英文**或**日文**的职场对话，并有效累积您的专属生词与实用句型！

---

## 🔑 第一步：设置与 API 启用 (非常重要)

默认情况下，系统使用"模拟对话体验模式"来展示基本流程。若要解锁强大的 AI 功能，请依照以下步骤设置：

1. 点击左侧菜单最下方的 **[设置与 API]**。
2. 选择您想使用的 **AI 服务商**，并填入对应的 API 密钥与模型名称：

> **💎 方案 A：Google Gemini (默认推荐)**
> 1. 前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 登入。
> 2. 点击 **"Create API key"** > **"Create API key in new project"**。
> 3. 复制以 **AIza** 开头的字符串，贴回本系统即可。
> *(注：Gemini API 目前对一般用户提供大方免费额度，免绑信用卡！)*

> **⚡ 方案 B：Groq (极速回复体验)**
> 1. 前往 [GroqConsole](https://console.groq.com/keys) 登入。
> 2. 点击 **"Create API Key"**。
> 3. 复制产生的密钥 `gsk_...` 贴回系统。
> 4. **模型名称 (Model ID)**：请确保填写正确的模型名称，默认推荐填写 `llama-3.1-8b-instant` 或 `llama-3.3-70b-versatile`，这能带来极速的体验！

3. 在同一页面中，您也可以设置您的**主类别**、**职务**、**语言程度**、以及您想要学习的**目标语言**。
4. 设置完成后点击"存储设置并返回"。

---

## 🗺️ 第二步：开始您的学习旅程

本系统的设计围绕着高效的学习循环：**"任务选取 ➔ 实战对话 ➔ 知识解析 ➔ 复习与导出"**。

### 1. 每日任务 (Dashboard)
- 进入 **[每日任务]**，系统会根据您的职务，列出 6 个不同难度的"情境任务"（从简单的工作回报到困难的技术谈判）。
- **开始练习**：点选任一卡片上的"开始练习"，即可进入专属该情境的对话。
- **✨ 自由对话模式 (Free Mode)**：不想受到主题限制？点击任务列表最上方的 **"✨ 自由对话模式"**，即可随心所欲地跟 AI 闲聊近况或探讨最新趋势，训练最真实的临场反应！

### 2. 对话练习模式 (Chat Practice)
- **口说输入**：点击"麦克风🎙️"按钮，直接用语音说出您的回复。
- **自动朗读**：AI 回复后，系统会自动朗读（可在设置中关闭）。若想再次聆听，可点击每则消息下方的"朗读"按钮。
- **💡 本情境推荐句型**：在聊天窗口上方，您可以展开面板，查看适合该情境的母语人士常用句型，试着把它们用在您的回复中！（*注：自由对话模式下将自动隐藏此面板*）
- **✨ AI 润饰建议**：即使是用中文构思，或是输入不完整的单词草稿，只要点击对话框旁的"魔法棒🪄"，AI 就会根据上下文，帮您翻译或修饰成最道地的母语说法。
- **🔍 当句解析与单词萃取**：遇到看不懂的 AI 回复？点击消息下方的"当句解析"，AI 会详细拆解句型文法，并自动为您萃取关键单词，收录至笔记本！
- **📥 导出与 AI 总评**：在与 AI 进行了一段对话后，点击画面上方的"导出"按钮，您可以选择直接导出对话记录 PDF，或是选择"导出并请 AI 解析"，让 AI 为您整场的口说表现、文法、语气打分数并给予总结建议！

### 3. 生词与句型笔记本 (Notebook)
- 在对话中被 AI 纠正的单词，或是透过"当句解析"萃取出的精华，都会自动收录于左侧的 **[生词笔记本]** 与 **[句型笔记本]** 中。
- 您可以随时点击发音按钮复习，或是点击"导出"将它们打包成 PDF 文件打印下来。

> [!WARNING]
> **⚠️ 数据存储注意事项**：您的笔记数据是存储在您的"浏览器本地端 (Local Storage)"。如果清除了浏览器的 Cookie 或缓存数据，笔记将会丢失！建议您**定期使用"导出"功能**，将辛苦收集的生词与句型备份为 PDF 文件。

### 4. 常用句型总览 (Patterns)
- 进入 **[常用句型]** 页面，系统会为您汇总当前情境的所有精选句型。
- **版本切换**：您可以在上方自由切换"基础教科书版 (01)"与"进阶商务版 (02)"，体验不同层次的表达方式。
- **代换练习**：点击任何一个句型卡片上的"进入代换练习"，系统会启动教练模式，给予您中文情境，训练您将单词套入句型中灵活运用！

## 💡 进阶技巧
- **选取文字解析**：在对话过程中，您可以直接"反白选取"AI 消息中的某个特定单词或短语，系统会跳出"标记解析"按钮，让您针对该字词进行深入学习。
- **文法纠错严格度**：若您只是想练胆量，可以在设置中选择"沟通为主"；若您即将面对重要面试或演示汇报，可切换至"超级严格"模式，AI 将会抓出您所有细微的不自然表达。
