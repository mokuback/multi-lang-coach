# Multi-Lang Coach에 오신 것을 환영합니다! 🚀

이것은 IT 전문가 및 비즈니스 담당자를 위해 특별히 설계된 "빅데이터 기반 음성 대화 코치"입니다.
여기에서 직무와 일일 임무를 기반으로 **영어** 또는 **일본어** 직장 대화를 연습하고, 자신만의 어휘와 실용적인 패턴을 효과적으로 축적할 수 있습니다!

---

## 🔑 1단계: 설정 & API 활성화 (매우 중요)

기본적으로 시스템은 기본 흐름을 보여주기 위해 "모의 대화 경험 모드"를 사용합니다. 강력한 AI 기능을 사용하려면 다음 단계에 따라 설정하세요:

1. 왼쪽 메뉴 하단의 **[설정 & API]** 를 클릭합니다.
2. 사용하려는 **AI 제공업체**를 선택하고 해당 API 키와 모델 이름(Model ID)을 입력합니다:

> **💎 옵션 A: Google Gemini (기본 권장)**
> 1. [Google AI Studio](https://aistudio.google.com/app/apikey)에 로그인합니다.
> 2. **"Create API key"** > **"Create API key in new project"** 를 클릭합니다.
> 3. **AIza**로 시작하는 문자열을 복사하여 이 시스템에 다시 붙여넣습니다.
> *(참고: Gemini API는 현재 일반 사용자에게 신용 카드 등록 없이 넉넉한 무료 할당량을 제공합니다!)*

> **⚡ 옵션 B: Groq (초고속 응답 경험)**
> 1. [GroqConsole](https://console.groq.com/keys)에 로그인합니다.
> 2. **"Create API Key"** 를 클릭합니다.
> 3. 생성된 키 `gsk_...` 를 복사하여 시스템에 다시 붙여넣습니다.
> 4. **모델명 (Model ID)**: 올바른 모델 이름을 입력했는지 확인하세요. 기본 권장 사항은 `llama-3.1-8b-instant` 또는 `llama-3.3-70b-versatile` 이며, 놀라운 속도를 경험할 수 있습니다!

3. 동일한 페이지에서 **기본 카테고리**, **주제 및 역할**, **언어 수준** 및 배우고 싶은 **목표 언어**를 설정할 수도 있습니다.
4. 설정이 완료되면 "저장 및 돌아가기"를 클릭합니다.

## 🗺️ 2단계: 학습 여정 시작

이 시스템의 설계는 효율적인 학습 주기인 **"작업 선택 ➔ 대화 연습 ➔ 지식 분석 ➔ 복습 및 내보내기"** 를 중심으로 이루어집니다.

### 1. 일일 임무 (Dashboard)
- **[일일 임무]** 로 들어가면 시스템이 귀하의 역할에 따라 난이도가 다른 6가지 "시나리오 임무"를 나열합니다 (간단한 업무 보고에서 어려운 기술 협상까지).
- **연습 시작**: 아무 카드에서나 "연습 시작"을 클릭하면 해당 시나리오 전용 대화로 들어갑니다.
- **✨ 자유 대화 모드**: 주제에 얽매이고 싶지 않으신가요? 임무 목록 상단의 **"✨ 자유 대화 모드"** 를 클릭하여 AI와 최근 상황이나 최신 트렌드에 대해 자유롭게 이야기하며 가장 현실적인 현장 반응을 훈련할 수 있습니다!

### 2. 대화 연습 모드
- **음성 입력**: "마이크 🎙️" 버튼을 클릭하여 음성으로 직접 답장을 말합니다.
- **자동 읽기**: AI가 응답한 후 시스템이 자동으로 읽어줍니다 (설정에서 끌 수 있음). 다시 듣고 싶다면 각 메시지 아래의 "읽기" 버튼을 클릭하세요.
- **💡 추천 패턴**: 채팅 창 상단에서 패널을 확장하여 해당 시나리오에 적합한 원어민이 자주 사용하는 패턴을 확인할 수 있습니다. 답장에 사용해 보세요! (*참고: 이 패널은 자유 대화 모드에서는 자동으로 숨겨집니다*)
- **✨ AI 개선 제안**: 모국어로 생각하거나 불완전한 단어 초안을 입력하더라도 대화 상자 옆의 "마법 지팡이 🪄"를 클릭하기만 하면 AI가 문맥에 따라 가장 원어민다운 표현으로 번역하거나 다듬어 줍니다.
- **🔍 문장 분석 및 단어 추출**: 이해할 수 없는 AI 응답을 발견하셨나요? 메시지 아래의 "문장 분석"을 클릭하면 AI가 문법을 자세히 분석하고 귀하를 위해 주요 어휘를 자동으로 추출하여 단어장에 저장합니다!
- **📥 내보내기 및 AI 평가**: AI와 대화를 나눈 후 화면 상단의 "내보내기" 버튼을 클릭합니다. 대화 기록을 직접 PDF로 내보내거나 "내보내기 & AI 분석 요청"을 선택하여 AI가 귀하의 전반적인 말하기 성능, 문법 및 어조를 평가하고 요약된 조언을 제공하도록 할 수 있습니다!

### 3. 단어장 및 패턴 노트
- 대화 중 AI가 수정해 준 단어나 "문장 분석"을 통해 추출된 핵심 내용은 왼쪽의 **[단어장]** 과 **[패턴 노트]** 에 자동으로 저장됩니다.
- 발음 버튼을 클릭하여 언제든지 복습하거나 "내보내기"를 클릭하여 PDF 파일로 압축하여 인쇄할 수 있습니다.

> [!WARNING]
> **⚠️ 데이터 저장 주의 사항**: 노트 데이터는 귀하의 "브라우저 로컬 저장소"에 저장됩니다. 브라우저의 쿠키나 캐시 데이터를 지우면 노트가 손실됩니다! 열심히 모은 어휘와 패턴을 PDF 파일로 백업하기 위해 **정기적으로 "내보내기" 기능을 사용하는 것**을 권장합니다.

### 4. 자주 쓰는 패턴
- **[자주 쓰는 패턴]** 페이지로 들어가면 시스템이 현재 시나리오에 대해 선택된 모든 패턴을 편집해 줍니다.
- **버전 전환**: 상단에서 "교과서 버전(초보자용)"과 "고급 비즈니스 버전(원어민 사고)" 간에 자유롭게 전환하여 다른 수준의 표현을 경험할 수 있습니다.
- **대체 연습**: 모든 패턴 카드의 "대체 연습"을 클릭하면 시스템이 코치 모드를 활성화하고 네이티브 시나리오를 제공하며 단어를 패턴에 유연하게 적용하도록 훈련합니다!

## 💡 고급 팁
- **선택한 텍스트 분석**: 대화 중에 AI 메시지의 특정 단어나 구를 직접 "강조 및 선택"할 수 있습니다. 시스템에 "분석됨" 버튼이 나타나 특정 용어를 심층적으로 학습할 수 있습니다.
- **문법 교정 엄격도**: 말하기 용기만 키우고 싶다면 설정에서 "의사소통 중심 (주요 오류만 교정)"을 선택할 수 있습니다. 중요한 인터뷰나 프레젠테이션을 앞두고 있다면 "매우 엄격 (모든 사소한 오류 교정)" 모드로 전환하면 AI가 미묘하게 부자연스러운 표현을 모두 잡아냅니다.

## 📱 웹사이트를 모바일 앱으로 설치 (다운로드 불필요)

이 시스템은 반응형 디자인을 지원합니다. 최상의 사용자 경험 (전체 화면, 브라우저 주소 표시줄 간섭 없음)을 위해 이 웹사이트를 휴대폰의 홈 화면에 추가하는 것을 권장합니다:

### 🍎 iOS (iPhone / iPad) 사용자:
1. **Safari** 브라우저를 사용하여 이 웹사이트를 엽니다.
2. 화면 하단의 **"공유 ⬆️"** 아이콘을 클릭합니다.
3. 아래로 스크롤하여 **"홈 화면에 추가"** 를 선택합니다.
4. 오른쪽 상단의 "추가"를 클릭하면 바탕 화면에 전용 앱 아이콘이 나타납니다!

### 🤖 Android 사용자:
1. **Chrome** 브라우저를 사용하여 이 웹사이트를 엽니다.
2. 화면 오른쪽 상단의 **"설정 (점 세 개 ⋮)"** 아이콘을 클릭합니다.
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 를 선택합니다.
4. 추가를 확인하면 휴대폰 바탕 화면에서 일반 앱처럼 직접 열 수 있습니다!
